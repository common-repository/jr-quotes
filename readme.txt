== Description ==

This plugin allows you to display geek quotes, pop quotes, general quotes, religious quotes and sci-fi quotes, all with a simple widget.

You can select the header of the widget, along with the type of quotes to display, through the admin panel. 



It's very easy to setup the plugin - Simply enable it, alter the settings and then enable the widget!

== Installation ==

1. Upload `jr-quotes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Quotes Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
